## Usage
```
# python fip.py
Usage: fip.py <ip> [<number_rand>]
```
## Example
```
# python fip.py 127.0.0.1 2
IP WITH RANDOM 2
dec8:   127.0.0.1
short:  127.1
dec32:  2130706433
hex8:   0x7f.0x0.0x0.0x1
hex32:  0x7f000001
oct8:   0177.0000.0000.0001
oct32:  017700000001
rand1:  127.0x0.0000.0001
rand2:  0x7f.0x0.0.0001
```
